# tmc-local-frontend

## Project setup
```
yarn global add @vue/cli
yarn global add storybook

yarn install
```

### Browse UI components with Storybook!
```
yarn run storybook
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
